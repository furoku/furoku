## 🤔 What is BananaX? BananaX가 뭐예요? BananaX 是什麼？

BananaX lives right inside your X (Twitter) compose flow. Write your post, click the 🍌 button, and **boom** — an AI-generated infographic appears, ready to attach.

**포스팅 흐름을 벗어나지 않고** 원클릭으로 이미지가 생성됩니다. 정말 편해요! 😎

**完全不需要離開發文流程**，一鍵就能生成圖片。超方便的！🚀

## 🔑 Getting Started 시작하기 開始使用

1. **Install BananaX** from the [Chrome Web Store](https://chromewebstore.google.com/detail/%F0%9F%8D%8C-bananax/nickihglllcjafklbpofgpcpoecacpji) 🍌
2. **Get a free API key** from [Google AI Studio](https://aistudio.google.com/apikey) 🔑
3. **Enter your API key** in BananaX settings ⚙️
4. **Go to X, write a post, click 🍌** — that's it! 🎉

> 설치하고, API 키 넣고, 바나나 버튼 누르면 끝! 진짜 간단해요 😆
