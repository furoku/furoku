📐 QUBO とは？

QUBO（Quadratic Unconstrained Binary Optimization）は、0 か 1 の変数だけを使って、ある目的関数を最小化する問題です。

最小化: f(x) = Σᵢ qᵢᵢxᵢ + Σᵢ<ⱼ qᵢⱼxᵢxⱼ
ただし xᵢ ∈ {0, 1}

多くの組み合わせ最適化問題（ナップサック、グラフカット、スケジューリングなど）をこの形に変換できます。量子コンピュータ（量子アニーリング）でも同じ定式化が使われます。

💡 ポイント

  • 線形項 qᵢᵢ: 変数 xᵢ を 1 にしたときのコスト
  • 二次項 qᵢⱼ: xᵢ と xⱼ が同時に 1 のときの追加コスト
  • 制約条件はペナルティ項として目的関数に組み込む

🔥 シミュレーテッドアニーリングとは？

金属の焼きなまし（アニーリング）にヒントを得た最適化手法です。高温から徐々に冷却することで、良い解を見つけます。

受理確率: P = exp(-ΔE / kT)
温度 kT は時間とともに減少 → kT × cl

🌡️ 温度の役割

  • 高温: 悪い変化も受け入れる → 広く探索
  • 低温: 良い変化のみ受け入れる → 解を洗練
  • ゆっくり冷やすほど良い解が見つかりやすい

⚙️ bqubo のパラメータ

  • kTinit: 初期温度（大きいほど探索範囲が広い）
  • cl: 冷却率（1に近いほどゆっくり冷却）
  • rpt: 各温度でのモンテカルロステップ数
  • kTF: 終了温度（これ以下で停止）

📝 QUBO定式化

各エッジ (i,j) について、カットされる ＝ xᵢ ≠ xⱼ。
最大化 → QUBO最小化に変換：

Q[i,j] = +2 (同じグループならペナルティ)
Q[i,i] -= degree(i) (各ノードの次数分)
