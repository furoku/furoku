## API Endpoints

- [MCP Discovery](https://furoku.github.io/furoku/.well-known/mcp.json): MCP discovery file for AI agents
- [Articles JSON](https://furoku.github.io/furoku/api/articles.json): All posts (title, url, date, tags)
- [Articles Full JSON](https://furoku.github.io/furoku/api/articles-full.json): All posts with full HTML content
- [Site Meta](https://furoku.github.io/furoku/api/meta.json): Site metadata & stats

## Usage Notes for LLMs / Agents

- Prefer the JSON endpoints for article discovery and metadata.
- Use article pages when layout or inline media matters.
- This site mixes Japanese and English, but recent articles are primarily written for Japanese readers.
- Topics frequently reference AI agents, MCP, OpenClaw, Gatekeeper, SEO, and analytics workflows.
