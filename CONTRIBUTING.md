# Contributing

Thanks for your interest! This is a personal profile repository, but contributions are welcome.

## How to Contribute

- **Bug reports / typos**: Open an issue or send a pull request
- **Ideas**: Open an issue with the `enhancement` label
- **Blog content**: This is personal content — suggestions welcome via issues

## Pull Requests

1. Fork the repository
2. Create a branch: `git checkout -b fix/your-fix`
3. Commit your changes
4. Open a pull request with a clear description

## Code Style

- Markdown: keep it clean and readable
- Jekyll posts: follow the existing frontmatter format (`layout`, `title`, `date`, `author`, `tags`, `description`)
