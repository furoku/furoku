# Security Policy

## Supported Versions

This is a personal profile and blog repository. There is no versioned software to patch.

## Reporting a Vulnerability

If you discover a security issue (e.g. exposed secrets, malicious content injected via a dependency):

1. **Do not open a public issue**
2. Email: `flock.h@gmail.com`
3. Include a description of the issue and steps to reproduce

I'll respond within 72 hours.
